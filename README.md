# DL4PDE Final Project

Final project for the course **Deep Learning for PDEs**.

This repository contains the source code and implementations for the four PDE problems investigated in the final project.

## Contents

- `Problem_A.ipynb` -- VPINN-based inverse problem for recovering the spatially varying Young's modulus in 1D elastostatics.
- `Problem_B.ipynb` -- Deep Ritz method for solving a heterogeneous Darcy flow problem.
- `Problem_C.ipynb` -- Fourier Neural Operator for learning the solution operator of a heterogeneous heat conduction problem.
- `Problem_D.ipynb` -- Physics-Informed Neural Operator with an FNO backbone for learning the solution operator of the viscous Burgers equation.

## Implementation

The numerical experiments and neural network models are implemented in Python using PyTorch.

Each Jupyter notebook contains the corresponding model definition, loss functions, training procedure, evaluation, and visualization of the numerical results.
