# DL4PDE Final Project

Final project for the course **Deep Learning for PDE**.

This repository contains the source code and implementations for the four problems investigated in the final project.

## Contents

- `Problem_A.ipynb` — Inverse problem for recovering Young's modulus in 1D elastostatics.
- `Problem_B.ipynb` — Deep learning based PDE problem.
- `Problem_C.ipynb` — Deep learning based PDE problem.
- `Problem_D.ipynb` — High-resolution PDE learning problem.

## Implementation

The numerical experiments and neural network models are implemented in Python using PyTorch.

Each Jupyter notebook contains the corresponding model definition, training procedure, evaluation, and visualization of the results.
